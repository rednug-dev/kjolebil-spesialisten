## Kom i gang

```bash
git clone git@github.com:<org>/kjolebil-spesialisten.git
cd kjolebil-spesialisten
pnpm install          # bruk pnpm 10+
cp .env.example .env.local
pnpm dev              # starter dev-server (Next.js + Turbopack)
```
